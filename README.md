# Chat TCP

## 📌 Descrição

Este projeto consiste no desenvolvimento de uma aplicação de chat utilizando o modelo Cliente-Servidor com comunicação através de Sockets TCP.

O projeto foi desenvolvido para a disciplina de Redes de Computadores da Universidade Federal da Paraíba (UFPB), com o objetivo de aplicar conceitos fundamentais de comunicação em redes, como sockets, protocolo TCP, endereçamento IP e portas.

---

## 🎯 Objetivos

- Desenvolver um servidor TCP.
- Desenvolver um cliente TCP.
- Permitir comunicação entre cliente e servidor.
- Compreender o funcionamento da arquitetura Cliente-Servidor.
- Aplicar os conceitos estudados na disciplina de Redes de Computadores.
